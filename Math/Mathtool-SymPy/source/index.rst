《SymPy 笔记》前言
==================================

SymPy 是 Python 的一个符号计算库．

本书是我学习 SymPy 的笔记，主要参考了 `官方教程 <https://docs.sympy.org/latest/tutorial/index.html>`_ ．在阅读本书之前，读者需要的知识有：

* Python 的基础语法
* 相关章节的数学知识

建议阅读、或可能有助于本笔记学习的知识有：

* NumPy 与 Matplotlib 库的使用
* SciPy 库的使用

本文撰写时的 SymPy 版本：

.. ipython:: python

    import sympy
    sympy.__version__
