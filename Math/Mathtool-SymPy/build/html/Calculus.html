2. 微积分与极限¶

符号计算自然不能少了微积分的内容．

In [1]: from sympy import *

2.1. 导数¶

函数的导数使用 diff(expr, var1, [order1,] var2, [order2], ... ) 或 f.diff(...) 来求得．其中，如果导数是一阶的，那么 order 参数可以省略．

下例分别求出了 与 ：

In [2]: x, y = symbols('x y')

In [3]: f = x**4 + 3 * x ** 2 * y ** 2 + y**4

In [4]: diff(f, x)
Out[4]: 4*x**3 + 6*x*y**2

In [5]: f
Out[5]: x**4 + 3*x**2*y**2 + y**4

In [6]: f.diff(x, 2, y)
Out[6]: 12*y

In [7]: f
Out[7]: x**4 + 3*x**2*y**2 + y**4

以上两种调用 diff 的方式并无差别．

2.2. 积分¶

不定积分，例如 的计算：

In [8]: integrate(exp(-x), x)
Out[8]: -exp(-x)

定积分的计算只需要将第二参数换成一个元组 (var, lower_limit, upper_limit) 即可，比如 ：

In [9]: integrate(exp(-x), (x, 0, oo))
Out[9]: 1

多重积分，比如 ：

In [10]: integrate(exp(-x**2 - y**2), (x, -oo, oo), (y, -oo, oo))
Out[10]: pi

2.3. 极限¶

用 limit(func, var, point, direction) 来计算极限，其中第四参数省略时表示双侧（即通常意义上的）极限，用 +/- 表示从正向/负向接近的一侧的极限：

In [11]: limit(sin(x)/x, x, 0)
Out[11]: 1

In [12]: limit(1/x, x, 0, '+')
Out[12]: oo

In [13]: limit(-1/x, x, 0)
Out[13]: -oo

2.4. 高阶项与 Taylor 展开¶

高阶项在 SymPy 中可以直接用 O(var**k) 的形式 [1] 写入表达式中，表示在变量趋向无穷时式中等于或高于 k 阶的项（因此，这些项的形式与系数在式中不再详细给出）：

In [14]: x**2 * O(x)
Out[14]: O(x**3)

# Might not be good if using non-poly terms
In [15]: sin(x) * O(x)
Out[15]: O(x**2)

In [16]: exp(x) * O(1)
Out[16]: O(1)

将表达式在某点附近作 Taylor 展开会涉及到上述的高阶记号．Taylor 展开可使用 expr.series(var, point, n) 函数：

In [17]: f = cos(x)

In [18]: f.series(x, 0, 6)  # omit x**k (k>=6) term
Out[18]: 1 - x**2/2 + x**4/24 + O(x**6)

可以用 removeO() 函数（移除大 O）来移除高阶项：

In [19]: f.series(x, 0, 6).removeO()
Out[19]: x**4/24 - x**2/2 + 1

注释

[1]数学中的大 O(x**k) 记号表示忽略 k 阶及 k 阶以上的高阶项；而计算机中的复杂度记号 O(n**k) 表示 n 趋向无穷时它渐进的速度，通常用于描述算法的时间或空间消耗与输入数据大小的关系．二者请勿混淆．
