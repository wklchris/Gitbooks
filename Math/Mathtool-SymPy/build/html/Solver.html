3. 解方程¶

SymPy 作为符号运算库，自然可以求解方程的解析解．

3.1. 通用求解器¶

通用求解函数是 solveset(eq, var, domain=S.Complexes)) ．其中，方程可以用上文介绍过的 Eq(LHS, RHS) 表示，也可以用 expr 代替以求解 expr=0 的方程：

In [1]: from sympy import *

In [2]: x, y = symbols('x y')

In [3]: solveset(Eq(x**2, -1), x)
Out[3]: {-I, I}

# Same equation as above
In [4]: solveset(x**2 + 1, x)
Out[4]: {-I, I}

其中，第三参数表示求解域：

# Exclude complex roots
In [5]: solveset((x**2 + 1) * (x - 1), x)
Out[5]: {1, -I, I}

In [6]: solveset((x**2 + 1) * (x - 1), x, domain=S.Reals)
Out[6]: {1}

除了上述虚数的情形，特殊的还有：

  • 在复数域上无解（返回 EmptySet ）
  • 解为全复数域
  • 方程在形式上无解析解（返回一个 ConditionSet ）
In [7]: solveset(exp(x), x)  # Solve exp(x) = 0
Out[7]: EmptySet()

In [8]: solveset(x - x, x)  # Solve x - x =0
Out[8]: S.Complexes

In [9]: solveset(x - sin(x), x)  # Try to solve x - sin(x) = 0
Out[9]: ConditionSet(x, Eq(x - sin(x), 0), Complexes(Reals x Reals, False))

3.2. 重根¶

注意， solveset(eq, var) 不会强调重根 ．如果想要计算重根，使用 roots(expr, var) 函数：

In [10]: f = x**2 * (x - 1)

In [11]: solveset(f, x)
Out[11]: {0, 1}

In [12]: roots(f, x)
Out[12]: {1: 1, 0: 2}

3.3. 线性方程组¶

线性求解器 linsolve(eqsystem, vars) 将在未来并入 solveset 函数中；但至少在目前的版本，仍需要调用 linsolve 来求解线性方程组．

下例求解了： 

In [13]: x, y = symbols('x y')

In [14]: linsolve([ x + y - 4, x - y + 2 ], (x, y))
Out[14]: {(1, 3)}

或者，以上方程组，我们也写成矩阵形式 ：

并用线性求解器的另一种输入格式 linsolve((A, b), vars) ：

In [15]: M = Matrix(( [1, 1, 4], [1, -1, -2] ))

In [16]: A = M[:, :-1]

In [17]: b = M[:, -1]

In [18]: A
Out[18]: 
Matrix([
[1,  1],
[1, -1]])

In [19]: b
Out[19]: 
Matrix([
[ 4],
[-2]])

# Format 1: M is a extended matrix: [A, b]
In [20]: linsolve(M, (x, y))
Out[20]: {(1, 3)}

# Format 2: Input A and b seperately
In [21]: linsolve((A, b), (x, y))
Out[21]: {(1, 3)}

最后，线性方程组可以是不定解的，例如两个方程构成的三元线性方程组：

In [22]: x, y, z = symbols('x:z')

In [23]: A = Matrix(( [1, 1, 0], [1, -1, 0] ))

In [24]: b = Matrix(( [4], [-2] ))

In [25]: linsolve((A, b), (x, y, z))
Out[25]: {(1, 3, z)}

3.4. 非线性方程组¶

用法与线性方程组类似，不过使用的是 nonlinsolve(eqsystem, vars) ：

In [26]: nonlinsolve([x*y - 1, x + y - 1], (x, y))
Out[26]: {(1/2 - sqrt(3)*I/2, 1/2 + sqrt(3)*I/2), (1/2 + sqrt(3)*I/2, 1/2 - sqrt(3)*I/2)}

注意，如果方程组中含有三角函数，请使用 solve() 而不是 nonlinsolve() ：

In [27]: solve([sin(x + y), cos(x - y)], (x, y))
Out[27]: [(-3*pi/4, 3*pi/4), (-pi/4, pi/4), (pi/4, 3*pi/4), (3*pi/4, pi/4)]

以上求得的含三角函数的方程组的解，并不是其所有的解．

3.5. 微分方程¶

利用 dsolve(eq, var) 可以解微分方程，需要先定义一个函数 f ：

In [28]: f = symbols('f', cls=Function)

In [29]: diffeq = Eq(f(x).diff(x, 2) + 12 * x**2 - 6 * x, 0)

In [30]: dsolve(diffeq, f(x))
Out[30]: Eq(f(x), C1 + C2*x - x**4 + x**3)
