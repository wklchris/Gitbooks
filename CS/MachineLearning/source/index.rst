《机器学习笔记》前言
==================================

本书是我学习机器学习的笔记，主要参考了：

* `Google 机器学习速成课程（Machine Learning Crash Course） <https://developers.google.com/machine-learning/crash-course/>`_ ．该网页也提供了简体中文的教程（左下角更改语言）；有趣的是，它的中文语音版本是由机器学习方法合成的．
* 周志华. *机器学习* . 清华大学出版社，2016.
* Géron A.  *Hands-on machine learning with Scikit-Learn and TensorFlow: concepts, tools, and techniques to build intelligent systems* . O'Reilly Media, Inc., 2017.

在阅读本书之前，读者需要的知识有：

* Python 的基础语法，以及 pandas 的使用经验
* 基础的理工科数学、统计学知识

建议阅读、或可能有助于本笔记学习的知识有：

* TensorFlow 的使用经验
