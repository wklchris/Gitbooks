《机器学习笔记》前言¶

SymPy 是 Python 的一个符号计算库．

本书是我学习机器学习的笔记，主要参考了 `Google 机器学习速成课程（Machine Learning Crash Course）<https://developers.google.com/machine-learning/crash-course/>`_ ．

在阅读本书之前，读者需要的知识有：

  • Python 的基础语法，以及 pandas 的使用经验
  • 基础的工科数学、统计学知识

建议阅读、或可能有助于本笔记学习的知识有：

  • TensorFlow 的使用经验
